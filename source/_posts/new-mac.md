# New Mac Setup

## Install Brew

`$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"`

## Install htop

`$ brew install htop`

## Install iterm2

Download the installer from [ITERM2 website](https://www.iterm2.com/).
